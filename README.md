# hotel-room-booking-project

A responsive Angular application for booking hotel rooms with real-time availability and filtering.

## Features
- View available hotel rooms with details
- Filter rooms by type (Single, Double, Suite)
- Book rooms with guest information
- Form validation and user feedback
- Toast notifications
- Responsive design for all devices
- localStorage persistence

## How to Run
1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm start`
4. Open browser to `http://localhost:4200`

## Running Tests
- Run all tests: `npm test`
- Tests include component and service coverage

## Design Decisions
- Used Angular Reactive Forms for better validation
- Modular component architecture for reusability
- Bootstrap utilities for responsive design
- RxJS for state management and mock APIs
- localStorage for data persistence
- Comprehensive testing strategy

## Tech Stack
- Angular 17+
- TypeScript
- Bootstrap 5
- RxJS
- Karma/Jasmine for testing
