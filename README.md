# hotel-room-booking-project

A responsive Angular application for booking hotel rooms with real-time availability and filtering.

## Features
- View available hotel rooms with details
- Filter rooms by type (Single, Double, Suite)
- Book rooms with guest information
- Form validation and user feedback
- Toast notifications
- Responsive design for all devices
- localStorage persistence

## Live Demo
- **StackBlitz**: https://stackblitz.com/edit/stackblitz-starters-7a6p8h2k
- **GitHub**: https://github.com/amirulafiq4work/hotel-room-booking-project

## How to Run and Test
1. Open the StackBlitz link above
2. The application will load automatically
3. Test all features:
   - View room list
   - Filter rooms by type
   - Book available rooms
   - See toast notifications
4. Your changes won't affect the original code (It will fork your changes)

## Design Decisions
- Used Angular Reactive Forms for better validation
- Modular component architecture for reusability
- Bootstrap utilities for responsive design
- RxJS for state management and mock APIs
- localStorage for data persistence
- Comprehensive testing strategy

## Tech Stack
- Angular 17+
- TypeScript
- Bootstrap 5
- RxJS
- Karma/Jasmine for testing
